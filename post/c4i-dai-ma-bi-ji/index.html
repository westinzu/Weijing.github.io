  • 正则是各取所需，非常容易 
- /^(?=.*?[A-Z]){3,4}$/     // 必须大写字母
- /^(?=.*?[A-Z])(?=.*?[0-9]){3,4}$/     // 必须大写字母，数字
- /^(?=.*?[A-Z])(?=.*?[0-9])(?=.*?[a-z]){3,4}$/     // 必须大写字母，数字，小写字幕
- /^(?=.*a)(?=.*b)(?=.*c)(?=.*d)(?=.*e).*$/ // 要用正则的断言,比如要出现a,b,c,d,e这5个字母,顺序任意,可以这样写
