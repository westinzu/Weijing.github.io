作者: 朱炜敬 时间: 2019年04月21日 17:12 描述: NUXT 初始化构建方案

image

  • 编译
    • webpack + babel ( ES6降级编译后自动打包)
    • vue-router (自动化注册路由)
    • vuex (记录单页面状态,组件共享数据)
    • Vue Server Renderer (配置权限)
  • 配置
    • mode spa srr (单页面渲染/服务器渲染)
    • Plugins UI组件库.工具类.http请求工具.错误监控.vue自定义指令等 需要在渲染页面前加载的JS (利用攻城道具,效率更高可指挥)
      • pug + stylus (css + html)
      • element (UI)
      • filter 过滤器 (date格式)
      • REM
    • css (通用样式配置/修改第三方UI样式)
    • layout 自定义错误页面
    • modules (nuxt增强功能)
    • proxyTable (代理配置)
    • styleResouces (全局加载css变量配置)
    • router (自定义规则/自动生成)
    • transition (动画)
  • 自定义拓展
    • ESlint + gitHost(校验代码质量,维护管理与同步习惯)
    • Fetch (Ajax)
    • javaScript 工具函数 (js脚手架)
