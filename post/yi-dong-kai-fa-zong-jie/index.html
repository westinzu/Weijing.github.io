移动开发总结

2019-05-22
  • 界面细节

    • 滚动不流畅 ⭕️

      • (在滚动元素上加)-webkit-overflow-scrolling

    • 异形屏兼容 ⭕️

      • 使用 cordova 或 中间服务 获取异形牌安全区尺寸

    • 禁止选择文字 ⭕️

        * {
    -webkit-user-select: nonei
    -khtml-user-select: nonei
    -moz-user-select: none_
    -ms-user-select; none_
    user-select: none;
  }

    • 图标尽力使用矢量图 ❇️

    • 阴影颜色设置 ⭕️

      • 这样 更符合物理世界的阴影效果 => 黑色透明实现 rgba(0, 0, 0, .3)

    • 0.5px的线宽 ⭕️

      • 由于现代分辨率高,手机端1px为逻辑像素而不是物理像素
      • IOS转换后能有很好的物理效果
      • Android 的解决办法: 给需要0.5px线的元素增加一个伪元素.然后伪元素的高度为1px.然后高度缩放一半

  )

    left: 0;
  width: 100_;
    height: 1px;
    background: var(--border);
  transform: scaleY(0.5) ;
  transform-origin: 50_ 100_;
  }
